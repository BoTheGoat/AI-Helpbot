# AI Helpdesk

A Java-based helpdesk ticketing system built to practice backend development, JDBC, MySQL, and software design patterns.

## Features

✅ Create Tickets

✅ View All Tickets

✅ Search Tickets by ID

✅ MySQL Database Integration

✅ DAO Pattern Implementation

## Tech Stack

- Java 8
- JDBC
- MySQL
- NetBeans
- Git/GitHub
